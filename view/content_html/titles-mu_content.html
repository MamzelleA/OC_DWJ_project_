Pour accéder au musée, vous devez vous munir d'un titre d'accès. Ils peuvent être classés ainsi :

LE BILLET

Pour les personnes devant s'acquitter du droit d'entrée. Il existe 2 types de billet :
Soit acheté en ligne sur le site du musée ou sur un site partenaire.
Il est horodaté.
Vous pouvez arriver dans la 1/2 heure précédent ou suivant l'horaire indiqué.
Vous avez un accès prioritaire.
Si vous l'avez acheté sur le site du musée, vous pouvez le modifier via votre compte sur le site de Billeterie avant le jour de votre visite.
Il n'est pas remboursable.
Sur le site officiel, il coûte 17€.

Soit acheté sur place.
Ce billet n'est valable que le jour de l'achat.
Il n'est ni échangeable ni remboursable.
Il coûte 15€.

Le billet vous donne accès aux collections 3 fois dans la journée c'est-à-dire que vous pouvez vous présenter 3 fois à un point de contrôle aux entrées des collections.

L'ACCES LIBRE

Il s'agit des porteurs de carte, voici une liste (non-exhaustive) des cartes :

  • Amis du Louvre
  • Louvre Professionnel.
  • Carte presse (nationale ou internationale)
  • Pass Education
  • ICOM / ICOMOS
  • Ministère de la culture
  • Partenariats d'entreprises

Vous avez un accès prioritaire.

L'ACCES GRATUIT

Il s'agit de personnes exonérées du droit d'entrée selon certains critères, cela concerne (liste non-exhaustive encore) :

  • les - 18 ans.
  • les - 26 ans résidents de l'Espace Economique Européen (EEE) [liste des pays]
  • les demandeurs d'emploi
  • les bénéficiaires de minima sociaux
  • les personnes en situation de handicap et leur accompagnateur

Evidemment vous devez avoir un justificatif.
A l'exception des personnes en situation de handicap, vous n'avez pas d'accès prioritaire. Mais, en vous rendant sur le site officiel du musée, vous avez la possibilité de réserver gratuitement un créneau de visite qui vous permettra de l'avoir.

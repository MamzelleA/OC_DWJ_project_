Vous avez la possibilité de voir la liste de tous les parcours ou de sélectionner un jour pour voir ceux qui sont réalisables.

Pour accéder aux détails d'un parcours et pouvoir le faire, vous devezêtre membre.

Quand vous cliquez sur "DETAILS", vous accédez à une "fiche parcours" (exemple ci-dessous).
Vous ne pouvez enregistrer un parcours que si vous avez sélectionnez un jour.
Pour pouvoir faire un parcours, vous devez l'enregistrer. Il sera alors faisable depuis la rubrique "VISITE".
